Author: Xingzhong Fan

Purpose:
Read mobile phone pedometer, accelerometer, gyroscope data in real time, and display it in the APP.

App.js:
The main App code load Pedometer data

App_Gyro.js:
The main App code load Gyroscope data

App_Acce.js:
The main App code load accelerometer data

app.json:
Expo version and information

package.json:
Expo pakage

Select a file in App.js, App_Gyro.js, App_Acce.js, and keep the others unchanged, and you can compile in Expo.
